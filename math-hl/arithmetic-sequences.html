Problem 1 — Basic terms and sum
Given an arithmetic sequence with first term a1 = 5 and common difference d = 3, find (a) the 20th term, and (b) the sum of the first 20 terms.
(a) nth term: a_n = a1 + (n-1)d = 5 + 19·3 = 5 + 57 = 62.
(b) Sum of first n terms: S_n = n*(a1 + a_n)/2 = 20*(5 + 62)/2 = 10*67 = 670.

Problem 2 — Determine a1 and d from sums
The sum of the first 10 terms of an arithmetic sequence is 150 and the sum of the first 20 terms is 420. Find a1 and d.
Let S_n = n/2 (2a1 + (n-1)d). S_10 = 10/2 (2a1 + 9d) = 5(2a1 + 9d) = 150 → 2a1 + 9d = 30. (1) S_20 = 10(2a1 + 19d) = 420 → 2a1 + 19d = 42. (2) Subtract (2)-(1): 10d = 12 → d = 1.2. Then 2a1 + 9(1.2) = 30 → 2a1 + 10.8 = 30 → 2a1 = 19.2 → a1 = 9.6.
Problem 3 — Smallest n for a sum bound
Find the smallest positive integer n such that the sum of the first n terms of the arithmetic sequence with a1 = 8 and d = -0.5 is less than 50.
S_n = n/2 (2a1 + (n-1)d) = n/2 (16 + (n-1)(-0.5)). We need S_n < 50. Solve n(16 - 0.5(n-1))/2 < 50 → n(16 - 0.5n + 0.5)/2 < 50 → n(16.5 - 0.5n) < 100. Rearranged: -0.5 n^2 + 16.5 n - 100 < 0 → multiply -2: n^2 -33 n + 200 > 0. Roots: n = [33 ± sqrt(33^2 - 800)]/2 = [33 ± sqrt(1089 - 800)]/2 = [33 ± sqrt(289)]/2 = [33 ± 17]/2 → n = 25 or n = 8. The quadratic is positive for n < 8 or n > 25. We need S_n < 50 which corresponds to n > 25 (since for small n the sum is increasing). So smallest integer n = 26.
