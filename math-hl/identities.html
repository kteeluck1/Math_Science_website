Problem 1 — Prove the following identity
(a) \(\displaystyle \frac{\sec\theta}{\sec\theta+\tan\theta} + \frac{\tan\theta}{\sec\theta-\tan\theta} = 1 + 2\tan^2\theta \)
Combine the two fractions and simplify. For clarity, write the result as display math:
\[\frac{\sec^2\theta + \tan^2\theta}{\sec^2\theta - \tan^2\theta}\] Using the Pythagorean identity \(\sec^2\theta = 1 + \tan^2\theta\) we obtain
\[\frac{1 + 2\tan^2\theta}{1} = 1 + 2\tan^2\theta,\] which proves the result.
